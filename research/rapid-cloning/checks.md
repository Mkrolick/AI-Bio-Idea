# Validation Checks - Rapid Gene Cloning

## Originality
- [ ] Searched literature for existing cell-free cloning approaches
- [ ] Confirmed this specific approach is not already published
- [ ] Identified what makes this distinct from Gibson Assembly, Golden Gate, etc.
- [ ] No existing commercial product does this exact thing

## Technical Tractability
- [ ] Core chemistry/biology is sound (cite specific papers)
- [ ] Required reagents/enzymes exist or are plausible to engineer
- [ ] No fundamental physical/chemical barriers identified
- [ ] Estimated cost per reaction is reasonable
- [ ] Timeline to prototype is < 2 years with existing technology

## Clarity & Completeness
- [ ] Idea can be explained in 2-3 paragraphs
- [ ] Concrete technical mechanism described (not hand-wavey)
- [ ] Speed improvement quantified (X days -> Y hours)
- [ ] Key risks acknowledged with mitigations

## Competition Readiness
- [ ] Passes the "would the judges find this boring?" test
- [ ] Not a general automation/cloud lab proposal
- [ ] Proposes a specific, concrete technical approach
- [ ] Addresses a widespread method (not niche)
