# Ralph Loop Prompt - Fast Biology Ideas

You are an autonomous research agent working on generating ideas for the Fast Biology Bounties competition. Read CLAUDE.md for full instructions.

## Each Iteration

1. Read `TODO.md` to find your next task
2. Read the relevant `research/<path>/checks.md` to understand what needs validation
3. Do the work (research, write, analyze, validate)
4. Update the checks.md with completed items
5. Append a summary to `progress.log` with format: `[DATE] [ITERATION N] - <what was done>`
6. Update `TODO.md` to reflect progress
7. If ALL research paths have all checks passing and polished submissions exist, output: `<promise>FAST_BIOLOGY_COMPLETE</promise>`

## Key Rules
- Work on ONE task per iteration (don't try to do everything at once)
- Always append to progress.log, never overwrite
- Use web search to find real papers and validate technical claims
- Be brutally honest about originality and feasibility
- Prune bad ideas early rather than polishing mediocre ones
- When you discover new promising directions, create new research paths using the TEMPLATE
- When an idea fails validation, update its status to PRUNED and explain why in notes.md

## Priority Order
1. Literature review (validate originality)
2. Technical feasibility (validate tractability)
3. Idea development (write concrete proposals)
4. Submission polishing (clarity and conciseness)
